# CALCULATOR APP 

## Description
Built with Laravel and Vue.js, it does simple math: add, subtract, multiply, and divide... It's user-friendly. Give it a try!
## Project setup
```
composer install
npm i
```

### Serve,compiles and hot-reloads for development
```
php artisan serve
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
